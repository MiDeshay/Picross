Picross!

What is Picross?

Picross is a grid-based logic puzzle game where the goal is to fill in all of the correct tiles on the grid to reveal a hidden image. To find the correct tiles, use the numbers along the top and side of the grid. The numbers near the grid follow basic rules:

  • An empty space, number, or set of numbers in a line are positioned next to every vertical and horizontal line in the grid and act as clues for that specific line.
  • The numbers along the top of the grid serve as clues for the vertical lines and the numbers along the side of the grid serve as clues for the horizontal lines.
  • An empty space next to a line of the grid indicates that there are no correct tiles (ie. tiles that need to be filled) in that line.
  • How many numbers there are next to a line of the grid indicates how many line segments there are.
  • A line segment is a connected group of correct tiles that is seperated from other groups of correct tiles by one or more incorrect tiles.
  • A number indicates the number of correct tiles in a segment.
  • The order that the numbers of a set appear in indicate the order in which each segment will appear.

Basic Strategy

Unlike traditional Picross, this version has a limited amount of incorrect answers. To avoid a game over, a basic strategy to solve a picross puzzle is to cross off or place an 'x' on incorrect tiles to make seeing correct tiles easier. Thinking logically and deducing which tiles need to be crossed off and which need to be filled is the core of gameplay. Take your time and have fun!
